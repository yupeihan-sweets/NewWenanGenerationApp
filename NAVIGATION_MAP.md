# 应用导航流程图

从主页开始，可以进入的所有页面及其导航关系。

## 📍 主页（首页）导航入口

### 1. 顶部区域
- **头像按钮** → `/profile` （个人资料页）

### 2. 搜索栏
- **搜索框** → `/templates` （灵感库页面）

### 3. 快速开始区域（4个快捷入口）
- **朋友圈** → `/generate/moments` （朋友圈生成器）
- **小红书** → `/generate/xiaohongshu` （小红书生成器）
- **短视频** → `/generate/video` （短视频脚本生成器）
- **全部** → `/scenarios` （全部场景页面）

### 4. Banner 卡片
- **"不知道写什么？试试我们的热门模板"** → `/templates/1` （模板详情页）

### 5. 最近创作区域
- **"查看全部"按钮** → `/records` （创作记录列表页）
- **最近创作卡片** → `/records/1` （记录详情页）

---

## 📱 Tab 导航页面（底部导航栏）

应用底部有 4 个 Tab，可以从任何页面直接切换：

1. **首页** (`/`) - 主页
2. **灵感** (`/templates`) - 灵感库
3. **记录** (`/records`) - 创作记录
4. **我的** (`/profile`) - 个人资料

---

## 🔄 页面详细导航关系

### 1. 首页 (`app/(tabs)/index.tsx`)
**可进入的页面：**
- `/profile` - 点击头像
- `/templates` - 点击搜索栏
- `/generate/moments` - 点击"朋友圈"快捷入口
- `/generate/xiaohongshu` - 点击"小红书"快捷入口
- `/generate/video` - 点击"短视频"快捷入口
- `/scenarios` - 点击"全部"快捷入口
- `/templates/1` - 点击 Banner 卡片
- `/records` - 点击"查看全部"
- `/records/1` - 点击最近创作卡片

### 2. 灵感库 (`app/(tabs)/templates.tsx`)
**可进入的页面：**
- `/templates/:id` - 点击任意模板卡片（如 `/templates/1`, `/templates/2` 等）

**功能：**
- 搜索模板
- 按分类筛选（全部、生活、职场、营销、情感、娱乐）

### 3. 创作记录 (`app/(tabs)/records.tsx`)
**可进入的页面：**
- `/records/:id` - 点击任意记录卡片（如 `/records/1`, `/records/2` 等）

**功能：**
- 查看所有创作记录
- 按时间线展示

### 4. 个人资料 (`app/(tabs)/profile.tsx`)
**可进入的页面：**
- 暂无（菜单项"我的收藏"未实现跳转，应该跳转到 `/favorites`）

**功能：**
- 显示用户信息
- 统计数据（生成次数、收藏模板、当前计划）
- 菜单项（我的收藏、会员中心、通用设置）

### 5. 朋友圈生成器 (`app/generate/moments.tsx`)
**可进入的页面：**
- `/result` - 点击"立即生成"按钮

**功能：**
- 输入今天发生的事情
- 选择心情
- 生成朋友圈文案

### 6. 小红书生成器 (`app/generate/xiaohongshu.tsx`)
**可进入的页面：**
- `/result` - 点击"创建专属笔记"按钮

**功能：**
- 输入产品/主题
- 输入卖点/关键词
- 选择目标人群（学生党、上班族、宝妈）

### 7. 短视频脚本生成器 (`app/generate/video.tsx`)
**可进入的页面：**
- `/result` - 点击"生成分镜脚本"按钮

**功能：**
- 输入视频主题
- 选择预估时长（15s、30s、45s、60s）
- 选择风格基调（搞笑、干货、治愈、悬疑、快节奏、唯美）

### 8. 全部场景 (`app/scenarios.tsx`)
**可进入的页面：**
- `/templates/1` - 点击任意场景卡片

**功能：**
- 展示所有可用场景（朋友圈、小红书、短视频、公众号、邮件、周报、其它）

### 9. 模板详情 (`app/templates/[id].tsx`)
**可进入的页面：**
- 暂无（详情页，通常作为终点页）

**功能：**
- 显示模板详细信息
- 展示模板介绍和使用说明

### 10. 记录详情 (`app/records/[id].tsx`)
**可进入的页面：**
- 暂无（详情页，通常作为终点页）

**功能：**
- 显示记录的完整内容
- 展示记录详情信息

### 11. 结果页 (`app/result.tsx`)
**可进入的页面：**
- 暂无（结果展示页，通常作为终点页）

**功能：**
- 显示生成结果
- 支持编辑结果
- 复制文案
- 重新生成

### 12. 我的收藏 (`app/favorites.tsx`)
**可进入的页面：**
- 暂无（从个人资料页的"我的收藏"菜单项应该可以进入，但代码中未实现）

**功能：**
- 展示收藏的文案
- 瀑布流布局展示

```
首页 → 小红书生成器 → 结果页

```


## 📝 注意事项

1. **Tab 导航**：底部 4 个 Tab（首页、灵感、记录、我的）可以从任何页面直接切换
2. **返回按钮**：大部分非 Tab 页面都有返回按钮，使用 `router.back()` 返回上一页
3. **动态路由**：
   - `/templates/:id` - 模板 ID 是动态的
   - `/records/:id` - 记录 ID 是动态的
4. **未实现的导航**：
   - 个人资料页的"我的收藏"菜单项应该跳转到 `/favorites`，但代码中未实现
5. **结果页**：所有生成器页面最终都会跳转到 `/result` 展示生成结果
